# УИвПРПО_pr4

Программа на Java для поиска и нормализации телефонных номеров в тексте.

## Возможности
- чтение текста из resources/input.txt
- поиск номеров с помощью регулярных выражений
- замена кода страны на +1
- приведение номера к единому формату
- запись результата в output.txt

### Настроен процесс непрерывной интеграции с использованием GitHub Actions.
